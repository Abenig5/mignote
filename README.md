# Catering Company Website

A modern catering company website built with Next.js, Turbopack, TypeScript, Tailwind CSS, and Prisma.

---

## Features

- Responsive modern UI
- Catering booking system
- Dynamic menu management
- Gallery showcase
- Testimonials
- Admin dashboard
- SEO optimized
- Fast performance

---

## Tech Stack

- Next.js
- Turbopack
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Framer Motion
- Shadcn UI

---

## Installation

```bash
npm install
```

---

## Development

```bash
npm run dev
```

Runs the Next.js development server with Turbopack. Use `npm run dev:webpack` if you need to compare behavior with webpack.

---

## Build

```bash
npm run build
```

Creates a production build with Turbopack. Use `npm run build:webpack` as a fallback.

---

## Folder Structure

```bash
src/
components/

types/
```

## Environment Variables

Create `.env.local`

```env
DATABASE_URL=
NEXT_PUBLIC_SITE_URL=
SMTP_USER=
SMTP_PASSWORD=
```
